# Static files for Siesta

Here we place static files for Siesta
to make it simpler to re-use pseudo's.
But also to shrink down the bandwidth usage
of the repo.

Add pseudo's as needed.


The directories contain files like this:

PD_fr_standard_04:
Pseudo-Dojo fully relativistic standard precision version 4.

PD_sr_standard_05:
Pseudo-Dojo scalar relativistic standard precision version 5.
